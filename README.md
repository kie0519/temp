# Git 练习项目

这是一个用于学习和练习 Git 操作的项目（已修复标题格式）。

## 项目简介

本项目用于实践以下 Git 技能：
- 基本的 Git 工作流（add, commit, push）
- 分支管理和合并
- 版本标签管理
- 团队协作流程

## 开始使用

```bash
# 克隆项目
git clone https://github.com/kie0519/temp.git

# 查看状态
git status

# 查看提交历史
git log
```
